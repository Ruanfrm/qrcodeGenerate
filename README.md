# Gerador de Qr Code

    projeto feito usando React Js.

## Tecnologias: 

    * Lib qrcode
    * lib react-qr-code
    * React + Vite

$ git clone https://github.com/Ruanfrm/qrcodeGenerate.git

* Inicializar
```bash
$ cd qrcode
$ npm install
$ npm install react-qr-code
$ npm install qrcode
$ npm run dev
```
